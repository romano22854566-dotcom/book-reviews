# Cайт для написания отзывов на книги

## Описание проекта

Веб-приложение, которое позволяет пользователям оценивать книги, оставлять отзывы и ознакамливаться с отзывами других пользователей.

## Основные функции
- Просмотр каталога книг с возможностью сортировки и фильтрации(по названию, id)
- Добавление книг в систему
## Реализованная архитектура (Слои)
Проект разделен на логические слои согласно принципам SOLID:
1. **Controller**: Обработка HTTP-запросов.
2. **Service**: Бизнес-логика приложения.
3. **Repository**: Взаимодействие с базой данных (JPA).
4. **Model/Entity**: Описание структуры таблицы БД.
5. **DTO (Data Transfer Object)**: Объект для передачи данных клиенту.
6. **Mapper**: Преобразование Entity в DTO.
## Технологии

* Backend - Java Spring Boot 
* Frontend - ещё не выбран
* Базы данных - PostgreSQL
